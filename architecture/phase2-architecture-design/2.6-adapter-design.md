# 2.6 Adapter Design

**Input:** 2.4 (6 ports) + 0.15 violations
**Protocol:** Evidence-First

## Design Principle

Adapters are thin delegates. No logic. No transformation. Just
`port.method = () => store.method()`.

## Adapters (6 — one per port)

### Adapter 1: MessagingAdapter
- **Implements:** MessagingPort
- **Delegates to:** useAppStore (chatSlice)
- **Methods:** 10 (same as port)
- **Location:** adapters/messagingAdapter.ts

### Adapter 2: NPCAdapter
- **Implements:** NPCCapability
- **Delegates to:** useAppStore (npcSlice + pressureSlice)
- **Methods:** 7
- **Location:** adapters/npcAdapter.ts

### Adapter 3: ArchiveAdapter
- **Implements:** ArchivePort
- **Delegates to:** useAppStore (archiveSlice + chatSlice divergence)
- **Methods:** 6
- **Note:** May need to import toggleChapter/toggleCategory from
  campaign-state service. This is acceptable — adapter is allowed
  to import domain utilities for state transformation.
- **Location:** adapters/archiveAdapter.ts

### Adapter 4: CampaignContextAdapter
- **Implements:** CampaignContextPort
- **Delegates to:** useAppStore (campaignSlice)
- **Methods:** 5
- **Location:** adapters/campaignContextAdapter.ts

### Adapter 5: SettingsAdapter
- **Implements:** SettingsPort
- **Delegates to:** useAppStore (settingsSlice)
- **Methods:** 3
- **Location:** adapters/settingsAdapter.ts

### Adapter 6: NotificationAdapter
- **Implements:** NotificationPort
- **Delegates to:** components/Toast (toast object)
- **Methods:** 4
- **Note:** This is the ONLY adapter that imports a component.
  This is acceptable because Toast is a leaf UI component with
  no domain knowledge — it's a rendering primitive.
- **Location:** adapters/notificationAdapter.ts

## Wiring

All 6 adapters are wired in main.tsx before React mounts:

```typescript
wireNotifications();
wireMessaging();
wireNPC();
wireArchive();
wireCampaignContext();
wireSettings();
```

## Adapter Rules (enforced)

1. Adapters MUST NOT contain business logic
2. Adapters MUST NOT import other adapters
3. Adapters MAY import store + ports (that's their job)
4. Adapters MAY import pure utilities (e.g., toggleChapter)
5. NotificationAdapter MAY import Toast (leaf component exception)

## Open Questions

None — adapter design is mechanical once ports are defined.
